How it works

  1. Set ReloadTemplates: true in pine.Config, then call render.Setup(app) — that's all.
  2. render.Setup installs the HTML engine and, because ReloadTemplates is true, registers a WebSocket endpoint at /__pine_reload and starts an fsnotify watcher on the ViewPath directory via websocket.Watch.
  3. The engine's Render() automatically injects a tiny <script> tag into every HTML response that connects to that endpoint.
  4. When you save a .html file, the watcher fires, engine.Rebuild() re-parses all templates, and a "reload" frame is broadcast to every connected tab.
  5. The browser receives the frame and calls location.reload().
